Cuatro encuentros para conocer estas tecnologías pilares del Desarrollo Web Front End.

Contenido

  • Introducción al desarrollo web: ¿Qué es un sitio web? / Arquitectura web cliente-servidor / ¿Qué son el front end y el back end? / HTML: ¿Qué es? ¿Para qué se usa? / Sintaxis / Estructura de un documento HTML / elementos HTML / Etiquetas basicas y atributos / Uso básico de editor de código y herramientas para desarrolladores del navegador
